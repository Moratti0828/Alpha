# 市场波动/回撤沟通脚本（安抚 + 纪律 + 不承诺）

关键词：波动，回撤，市场大跌，安抚，纪律，定投，再平衡，风险揭示

## 适用场景
- 用户说“最近跌了很多”“我扛不住了”“要不要清仓”“要不要加仓”。

## 推荐话术结构
1) 共情 + 风险提示  
- “我理解回撤会带来压力。需要先提醒：市场短期波动不可预测，基金/ETF也可能回撤。”

2) 关键判断：钱是否短期要用、仓位是否超承受  
- “这笔钱 6-12 个月内要用吗？当前权益仓位大概多少？最大能接受多大回撤？”

3) 给纪律方案（不讲点位）
- 若超承受：分批把仓位调回目标区间（降低被动卖出风险）
- 若长期资金且配置合理：保持分批/定投节奏，按规则再平衡，避免情绪化操作

## 禁写提醒
- 不要说“马上就会反弹”“一定能回本”
- 不要给具体点位/日期/涨跌幅预测
