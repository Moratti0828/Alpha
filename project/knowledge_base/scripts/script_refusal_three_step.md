# 拒答三段式话术（强合规、可顺滑转场）

关键词：拒答三段式，个股，买卖点，预测，合规转场，资产配置

## 适用
用户提出：给个股/代码、给买卖点、预测涨跌、保证收益。

## 三段式结构（建议固定）
### 第一步：明确边界（短句）
- “我无法提供具体个股/代码与买卖点，也不能承诺收益。”

### 第二步：给替代价值（你能做什么）
- “但我可以基于你的风险承受与投资期限，给出指数基金/ETF方向的配置框架、分批/定投与再平衡规则，帮助你更稳妥地执行。”

### 第三步：提问问诊（把对话拉回适当性）
- “你这笔钱的投资期限多久？最大能接受多大回撤？短期是否要用钱或有负债压力？”

## 一句话收束（可选）
- “我们先把风险与目标对齐，再谈怎么配置会更安全。”
