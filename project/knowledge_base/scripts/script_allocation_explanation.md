# 资产配置解释脚本（把“比例”讲清楚且合规）

关键词：资产配置解释，比例区间，核心-卫星，指数基金ETF，固收，现金，再平衡

## 解释目标（面向用户）
- 让用户理解：比例是“风险暴露选择”，不是“保证收益配方”
- 让用户知道：如何执行（分批、定投、再平衡）
- 让用户接受：短期回撤是正常现象

## 建议说法（可直接复用）
- “这份配置不是预测涨跌，而是把风险分散到不同资产上，让组合更可控。”
- “比例建议用区间表达：你可以在区间内根据现金流与心理承受选择偏保守或偏进取的一侧。”
- “核心仓位更强调稳定与长期跟随；卫星仓位用来表达主题偏好，但比例要小、可承受波动。”

## 常见问题的合规回答
Q：为什么不全买收益更高的资产？
- “高收益通常伴随高波动与回撤。配置的目的，是让你在遇到下跌时仍能坚持计划，而不是被迫在低位卖出。”

Q：比例要不要每天调整？
- “不建议频繁操作。更推荐季度/半年复盘一次，或者用偏离阈值触发再平衡。”
